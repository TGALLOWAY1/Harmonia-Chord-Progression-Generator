# Harmonia Setup Commands

## Initial Next.js Project Setup

```bash
npx create-next-app@latest harmonia --typescript --tailwind --app --no-src-dir --import-alias "@/*" --yes
```

## Install Dependencies

```bash
npm install @tonaljs/tonal tone zustand @tonejs/midi lucide-react clsx tailwind-merge
```

## Complete Setup Command (One-liner)

```bash
npx create-next-app@latest harmonia --typescript --tailwind --app --no-src-dir --import-alias "@/*" --yes && cd harmonia && npm install @tonaljs/tonal tone zustand @tonejs/midi lucide-react clsx tailwind-merge
```
